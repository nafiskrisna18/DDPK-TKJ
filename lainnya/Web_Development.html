1. Persiapan Tools & Lingkungan Kerja

Sebelum coding, seorang developer harus menyiapkan "senjata" perangnya.

Menginstal Visual Studio Code

VS Code adalah text editor paling populer saat ini karena ringan dan kaya fitur.

1

Unduh Installer

Kunjungi code.visualstudio.com dan unduh versi sesuai OS (Windows/Mac/Linux).

2

Instalasi

Jalankan installer. Centang opsi "Add to PATH" agar mudah diakses dari command prompt.

3

Pasang Ekstensi Wajib

Buka menu Extensions (Ctrl+Shift+X). Cari dan instal: Live Server (untuk preview real-time) dan Prettier (untuk merapikan kode).

2. Memahami HTML dan CSS

Analogi: HTML adalah kerangka manusia, CSS adalah pakaian dan riasannya.

HTML (HyperText Markup Language)

Menggunakan "Tags" untuk mendefinisikan elemen. Contoh struktur dasar:

<!DOCTYPE html>
<html>
  <head>
    <title>Judul Halaman</title>
  </head>
  <body>
    <h1>Halo Dunia!</h1>
    <p>Ini paragraf.</p>
  </body>
</html>

CSS (Cascading Style Sheets)

Menggunakan "Selectors" untuk memilih elemen HTML dan memberinya gaya.

/* Selector: h1 */
h1 {
  color: blue;
  font-size: 24px;
  text-align: center;
}

CSS Playground

div#box

Klik tombol di atas untuk menerapkan CSS ke kotak.

3. JavaScript pada Web

Membuat halaman web berinteraksi dengan pengguna (logika).

Variabel

Tempat menyimpan data.

let nama = "Budi"; const pi = 3.14;

Fungsi

Kumpulan perintah yang bisa dipanggil ulang.

function sapa() { alert("Halo!"); }

DOM Manipulation

Mengubah HTML/CSS lewat JS.

document.getElementById("id").style.color = "red";

Rangkuman, Soal & Praktikum

Uji pemahaman dan praktikkan keterampilanmu.

Rangkuman Materi

1. Apa itu Web Development?
Web Development adalah proses pembuatan dan pemeliharaan situs web. Terbagi menjadi Frontend (tampilan/klien) dan Backend (server/database). Tiga pilar utamanya adalah HTML (struktur), CSS (tampilan), dan JavaScript (interaksi).
2. Peran Tools (VS Code & Github)
Visual Studio Code adalah editor kode tempat kita menulis program. Github adalah platform version control berbasis cloud untuk menyimpan riwayat kode dan berkolaborasi dengan tim.

Soal Essay (Latihan)

1. Jelaskan perbedaan mendasar antara HTML dan CSS!

Jawaban: HTML (HyperText Markup Language) digunakan untuk membuat struktur dan konten halaman (seperti teks, gambar), sedangkan CSS (Cascading Style Sheets) digunakan untuk mengatur tampilan visual (seperti warna, font, layout) dari struktur tersebut.

2. Apa fungsi JavaScript dalam sebuah website?

Jawaban: JavaScript berfungsi untuk membuat halaman web menjadi interaktif dan dinamis. Contohnya: membuat animasi, memproses form, mengambil data dari server tanpa reload, dan merespons klik pengguna.

3. Mengapa kita perlu menggunakan Github?

Jawaban: Untuk menyimpan kode secara cloud (backup), melacak perubahan kode (version control), dan memudahkan kolaborasi tim dalam satu proyek yang sama.

4. Sebutkan 3 ekstensi VS Code yang membantu web developer!

Jawaban: 1. Live Server (untuk preview real-time), 2. Prettier (untuk merapikan format kode), 3. Auto Close Tag (untuk menutup tag HTML otomatis).

5. Apa yang dimaksud dengan Responsive Design?

Jawaban: Responsive Design adalah pendekatan desain web yang membuat tampilan website dapat menyesuaikan diri secara otomatis dengan ukuran layar perangkat pengguna (HP, Tablet, Laptop) agar tetap mudah dibaca dan digunakan.

Praktikum: Membuat Portfolio Pribadi

Tugas Akhir

Instruksi Kerja:

  • Buat folder proyek baru bernama portfolio-saya.
  • Buat file index.html. Gunakan tag <h1> untuk namamu dan <img> untuk fotomu.
  • Buat file style.css. Ubah warna background body dan font. Hubungkan ke HTML dengan <link>.
  • Tambahkan tombol "Hubungi Saya" yang ketika diklik memunculkan alert('Terima kasih!') menggunakan JavaScript.
  • Push proyek tersebut ke repository Github barumu.
