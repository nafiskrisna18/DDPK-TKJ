Sistem Layanan Jaringan

Jaringan komputer menyediakan berbagai layanan untuk mempermudah komunikasi dan berbagi sumber daya. Berikut adalah beberapa layanan yang paling umum.

DNS (Domain Name System)

Berfungsi menerjemahkan nama domain (contoh: google.com) menjadi alamat IP (contoh: 172.217.170.142) yang dapat dimengerti oleh komputer. Ibarat buku telepon untuk internet.

DHCP (Dynamic Host Configuration Protocol)

Memberikan alamat IP secara otomatis kepada perangkat yang terhubung ke jaringan. Ini menyederhanakan administrasi jaringan karena tidak perlu mengatur IP setiap perangkat secara manual.

HTTP/HTTPS (Hypertext Transfer Protocol/Secure)

Protokol yang digunakan untuk mentransfer data di World Wide Web (WWW). HTTPS adalah versi amannya yang mengenkripsi data, ditandai dengan ikon gembok di browser.

FTP (File Transfer Protocol)

Digunakan untuk mentransfer file dari satu komputer ke komputer lain melalui jaringan. Sering dipakai untuk mengunggah file website ke server hosting.

Email Services (SMTP, POP3, IMAP)

Terdiri dari beberapa protokol: SMTP untuk mengirim email, POP3 dan IMAP untuk menerima dan menyimpan email di perangkat klien.

Proxy Server

Bertindak sebagai perantara antara pengguna dan internet. Fungsinya bisa untuk filtering konten, meningkatkan keamanan, atau mempercepat akses dengan caching.

Sistem Keamanan Jaringan

Keamanan jaringan sangat penting untuk melindungi data dari akses tidak sah. Ini melibatkan perlindungan terhadap berbagai ancaman. Berikut adalah beberapa ancaman umum dan cara penanggulangannya.

Rangkuman

TCP/IP adalah model 4 lapis (Network Access, Internet, Transport, Application) yang menjadi dasar komunikasi di internet, memastikan data terkirim dengan benar dan sampai ke aplikasi yang tepat.

Layanan Jaringan seperti DNS, DHCP, dan HTTP adalah servis-servis penting yang berjalan di atas TCP/IP untuk membuat jaringan lebih fungsional dan mudah digunakan, misalnya menerjemahkan nama domain dan memberikan IP otomatis.

Keamanan Jaringan adalah upaya melindungi jaringan dari ancaman seperti malware, phishing, dan DDoS. Solusinya meliputi penggunaan Antivirus, Firewall, Enkripsi, dan kesadaran pengguna.

Sistem Transmisi sangat beragam. Sistem Seluler menyediakan mobilitas luas. WLAN (Wi-Fi) ideal untuk akses nirkabel lokal. Sistem Optik (Fiber Optic) menawarkan kecepatan dan kapasitas tertinggi untuk backbone. Microwave dan VSAT digunakan untuk komunikasi jarak jauh di mana kabel tidak praktis.

Setiap teknologi transmisi memiliki keunggulan dan kelemahan dalam hal kecepatan, jangkauan, biaya, dan keandalan, sehingga pemilihannya tergantung pada kebutuhan spesifik aplikasi.

1. Jelaskan fungsi dari setiap lapisan pada model TCP/IP, mulai dari lapisan terbawah hingga teratas!

Network Access Layer: Bertanggung jawab untuk mengirimkan data melalui media fisik jaringan (misalnya kabel Ethernet, Wi-Fi).
Internet Layer: Bertugas untuk pengalamatan logis (IP Address) dan routing (menentukan jalur terbaik) paket data di seluruh jaringan.
Transport Layer: Menyediakan komunikasi antar proses (aplikasi) dan memastikan pengiriman data yang andal (menggunakan TCP) atau cepat (menggunakan UDP).
Application Layer: Menyediakan antarmuka bagi aplikasi pengguna untuk mengakses layanan jaringan (misalnya HTTP untuk browsing, SMTP untuk email).

2. Apa perbedaan mendasar antara layanan DNS dan DHCP dalam sebuah jaringan komputer? Mengapa keduanya penting?

Perbedaan: DNS menerjemahkan nama domain menjadi alamat IP, sementara DHCP memberikan alamat IP secara otomatis ke perangkat. DNS ibarat "buku telepon", DHCP ibarat "petugas yang memberikan nomor rumah".
Pentingnya: Keduanya penting untuk otomasi dan kemudahan penggunaan. Tanpa DNS, kita harus menghafal alamat IP. Tanpa DHCP, administrator harus mengatur IP setiap perangkat secara manual, yang tidak efisien.

3. Jelaskan tiga jenis ancaman keamanan jaringan yang berbeda dan sebutkan satu cara untuk menanggulangi masing-masing ancaman tersebut!

1. Ancaman: Malware (misal: Ransomware) - Perangkat lunak jahat yang mengenkripsi data dan meminta tebusan. Solusi: Menggunakan Antivirus yang ter-update dan melakukan backup data secara berkala.
2. Ancaman: Phishing - Upaya menipu pengguna untuk mendapatkan informasi sensitif (seperti password) dengan menyamar sebagai pihak tepercaya. Solusi: Edukasi pengguna untuk tidak mengklik link atau lampiran yang mencurigakan.
3. Ancaman: Serangan DDoS - Membanjiri server dengan lalu lintas data palsu sehingga layanan menjadi tidak dapat diakses. Solusi: Menggunakan layanan proteksi DDoS atau mengkonfigurasi Firewall untuk memfilter lalu lintas berbahaya.

4. Bandingkan sistem transmisi Fiber Optik dengan sistem WLAN (Wi-Fi) berdasarkan kecepatan, jangkauan, dan mobilitas!

Fiber Optik:
- Kecepatan: Sangat tinggi (Gbps hingga Tbps).
- Jangkauan: Sangat jauh (puluhan hingga ribuan kilometer).
- Mobilitas: Rendah (statis, karena menggunakan kabel fisik).
WLAN (Wi-Fi):
- Kecepatan: Cukup tinggi (Mbps hingga Gbps), tetapi lebih rendah dari Fiber Optik.
- Jangkauan: Terbatas (beberapa puluh meter).
- Mobilitas: Tinggi (pengguna bisa bergerak bebas dalam area jangkauan).

5. Dalam situasi apa penggunaan sistem VSAT IP lebih diutamakan dibandingkan menggunakan jaringan seluler atau kabel?

Penggunaan sistem VSAT IP lebih diutamakan di lokasi-lokasi terpencil atau rural di mana infrastruktur jaringan kabel (seperti fiber optik atau tembaga) tidak tersedia dan sinyal jaringan seluler juga lemah atau tidak ada sama sekali. Contohnya adalah di daerah pertambangan, perkebunan di pedalaman, anjungan lepas pantai, atau untuk menyediakan konektivitas internet di desa-desa terpencil.
